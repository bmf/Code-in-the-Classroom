• Level 1: Alphabets

    Match alphabets to items starting with that letter. This is the first level teaching tool to start the children with understanding alphabet shapes.

    Each game will teach 3 alphabets. The order can be set to be sequential from A to Z or set to random, once the child has some knowledge of the alphabets.
  • Level 2: Words

    Match words to items corresponding to that word. This is the second level teaching tool to start the children with understanding full words.

    Each game will teach 3 words. The order can be set to be sequential from A to Z or set to random, once the child has some knowledge of the words.
  • Level 3: Phrases

    Match multiple words to images that describe the image. This is the third level teaching tool to start the children with understanding describing of more complex images.

    Each game will teach 2 phrases per graphic. The order can be set to be sequential from A to Z or set to random, once the child has some knowledge of the phrases.
